== Description ==

CSS Filter Generator is a WordPress plugin designed to calculate CSS filter values based on HEX color codes. This plugin is particularly useful for designers and developers who need to apply specific color filters to images or elements on their WordPress site.

== Installation ==

    Upload the plugin files to the /wp-content/plugins/custom-color-filter directory, or install the plugin through the WordPress plugins screen directly.
    Activate the plugin through the 'Plugins' screen in WordPress.
    Use the Settings->CSS Filter Generator screen to configure the plugin.

== Usage ==

    Go to Settings->CSS Filter Generator in your WordPress admin panel.
    Enter a HEX color code in the provided field (e.g., #ff0000).
    Click the "Calculate" button to generate the CSS filter value.
    The calculated filter value will be displayed on the screen, which you can then use in your CSS.

== Frequently Asked Questions ==

= Can I use this plugin to calculate filter values for any HEX color code? =

Yes, you can use any valid HEX color code to calculate the corresponding CSS filter value.
